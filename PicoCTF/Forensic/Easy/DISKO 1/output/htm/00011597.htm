In this pane, remote databases from dbhub.io website can be added to DB Browser for SQLite. First you need an identity:

  1. Login to the dbhub.io website (use your GitHub credentials or whatever you want)
  2. Click the button to "Generate client certificate" (that's your identity). That'll give you a certificate file (save it to your local disk).
  3. Go to the Remote tab in DB Browser for SQLite Preferences. Click the button to add a new certificate to DB Browser for SQLite and choose the just downloaded certificate file.

Now the Remote panel shows your identity and you can add remote databases.
